Description

我们知道利用函数交换2个数，如果采用值传递是实现不了的，那么怎么办呢？我们不是刚学过指针吧？对，就用指针来做。

Input

多组测试数据。每组测试数据输入2个整数。

Output

对于每组测试数据，从大到小输出数

Sample Input


2 3
4 5
Sample Output


3 2
5 4

#include <stdio.h> 
void swap(int *p1,int *p2) 
{ 
int t;
t=*p1;
*p1=*p2;
*p2=t;
} 
int main() 
{ int a,b; 
int *pointer_1,*pointer_2; 
while(scanf("%d%d",&a,&b)!=EOF) 
{ 
pointer_1=&a;
pointer_2=&b; 
if(a < b) 
{ 
swap(pointer_1,pointer_2);
} 
printf("%d %d\n",a,b); 
} 
}
